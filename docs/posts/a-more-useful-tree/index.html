A more useful `tree`

25 Sep, 2025

While Linux does have the tree command, if you’re doing programming, you probably don’t want results in folders like node_modules, dist, etc…, so here is a useful little alias I have in my ~/.bash_aliases:

 1# tree alias that excludes common directories
 2treen() {
 3    local ignore='node_modules|__pycache__|*.pyc|.git|.next|dist|build|
 4    coverage|.cache|.venv|venv|bin|obj|packages|.vs|*.user|*.suo'
 5    local args=(-I "$ignore")
 6    if [ $# -gt 0 ]; then
 7        args+=(-P "$1" --prune)
 8        shift
 9    fi
10    command tree "${args[@]}" "$@"
11}

You can use it like so: treen **/*.ts
